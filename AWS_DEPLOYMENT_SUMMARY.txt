WHAT YOU NEED TO DO FOR AWS
============================

STEP 1: Get AWS Credentials
   - AWS_ACCESS_KEY_ID
   - AWS_SECRET_ACCESS_KEY
   - Available from your AWS IAM console

STEP 2: Add to GitHub Repository Secrets
   Go to: https://github.com/YOUR_REPO/settings/secrets/actions
   
   Add these 4 secrets:
   - AWS_ACCESS_KEY_ID (your access key)
   - AWS_SECRET_ACCESS_KEY (your secret key)
   - AWS_ACCOUNT_ID: 626216981288
   - FRED_API_KEY (optional, for economic data)

STEP 3: Trigger Bootstrap Workflow
   Go to: Actions > Bootstrap OIDC Provider & Deploy Role
   Click: "Run workflow" button
   Wait: ~2 minutes
   
   This creates:
   - GitHub Actions OIDC provider
   - GitHubActionsDeployRole with AdministratorAccess
   
   After this, all future deployments use OIDC (no static keys exposed)

STEP 4: Deploy Infrastructure
   Go to: Actions > Deploy Infrastructure
   Click: "Run workflow" button
   Wait: ~15-20 minutes
   
   This creates:
   - VPC with public/private subnets
   - RDS PostgreSQL database
   - ECS cluster for data loaders
   - CloudWatch monitoring

STEP 5: Run Data Loaders (in phases)
   Phase 1 (10 min): Core metadata
   - Go to: Actions > Data Loaders Pipeline
   - Input: "stocksymbols,dailycompanydata,marketindices"
   - Click: "Run workflow"
   
   Phase 2 (30 min): Price history
   - Input: "pricedaily,priceweekly,pricemonthly,etfpricedaily,etfpriceweekly"
   
   Phase 3 (20 min): Trading signals
   - Input: "buyselldaily,buysellweekly,buysellmonthly,buysell_etf_daily,buysell_etf_weekly"
   
   Phase 4-5: Fundamentals, earnings, scores
   - Run remaining loaders in 5-loader batches

TOTAL TIME: ~2 hours from start to fully loaded system
