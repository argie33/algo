ESSENTIAL CORE (6 LOADERS - MUST RUN)
═══════════════════════════════════════════════════════════════════════════════

1. loadstocksymbols.py
   What: Load all stock symbols from NASDAQ/NYSE
   Source: nasdaq-symbols, yfinance
   Creates: stock_symbols table
   Run order: FIRST (everything else needs this)
   Time: ~5 minutes
   Must run: ✅ YES

2. loadpricedaily.py
   What: Download daily price history (OHLCV) for all stocks
   Source: yfinance (downloads historical data)
   Creates: price_daily table (symbol, date, open, high, low, close, volume)
   Run order: SECOND (after symbols)
   Time: VERY LONG - several hours due to yfinance rate limits
   Current: RUNNING NOW at batch 36/994
   Must run: ✅ YES - CORE DATA
   Problem: yfinance has rate limits - fetches only ~5 symbols per request

3. loadstockscores.py
   What: Calculate technical scores (momentum, value, quality, growth, etc.)
   Source: Reads from price_daily + technical indicators
   Creates: stock_scores table
   Dependencies: loadpricedaily (needs price data)
   Time: 30-60 minutes
   Must run: ✅ YES - NEEDED FOR SIGNALS
   Notes: Generates composite scores used in Buy/Sell signal generation

4. loadbuyselldaily.py
   What: Generate daily Buy/Sell trading signals
   Source: price_daily + stock_scores
   Creates: buy_sell_daily (signal, buylevel, stoplevel, date, symbol)
   Dependencies: loadstockscores
   Time: 1-2 hours
   Must run: ✅ YES - THIS IS YOUR MAIN SIGNAL
   Notes: Actual trading signals based on technical indicators

5. loadbuysellweekly.py
   What: Aggregate daily signals to weekly timeframe
   Source: buy_sell_daily
   Creates: buy_sell_weekly table
   Time: ~10 minutes
   Must run: ❓ ONLY if you trade weekly charts

6. loadbuysellmonthly.py
   What: Aggregate daily signals to monthly timeframe
   Source: buy_sell_daily
   Creates: buy_sell_monthly table
   Time: ~10 minutes
   Must run: ❓ ONLY if you trade monthly charts
