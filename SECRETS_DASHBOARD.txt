================================================================================
                          TROUBLESHOOTING GUIDE
================================================================================

If GitHub Actions fails after rotating keys:
  1. Check GitHub Actions logs for "AccessDenied" error
  2. New key likely missing IAM permissions
  3. Restore old keys temporarily: gh secret set AWS_ACCESS_KEY_ID --body "OLD_KEY"
  4. Debug IAM permissions on new key
  5. Try rotation again

If Dashboard shows "Data not available":
  1. Restart dev_server: python3 api-pkg/dev_server.py
  2. Restart dashboard: python3 -m dashboard --local

If Secrets Manager operations fail:
  1. Check IAM user has required permissions
  2. Contact AWS admin if needed
