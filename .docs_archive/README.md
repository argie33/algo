# Documentation Archive

Historical docs from past sessions. Archived because:
- They document completed work (see git log instead)
- They contain dated snapshots that rot over time
- Current state is in code + STATUS.md + git commits
- They bloat CLAUDE.md context

Includes: status reports, audit reports, completion summaries, session notes.

To restore a file: `git checkout <commit> -- path/to/file.md`
